VELAS-10: Integration Package - START.bat и Live Engine

Добавлено:
- START.bat — главный лаунчер системы с меню (9 опций)
- scripts/init_database.py — инициализация БД с тестовыми данными
- scripts/download_history.py — скачивание свечей с Binance (20 пар × 3 TF)
- config/config.example.yaml — пример конфигурации с комментариями
- backend/live/engine.py — Live Trading Engine с WebSocket
- backend/data/binance_ws.py — Binance WebSocket клиент
- backend/telegram/bot.py — Telegram notifier с Cornix форматом
- backend/core/velas_core.py — Индикатор Velas (каналы, ATR)
- backend/core/signals.py — Генератор сигналов с фильтрами
- backend/portfolio/manager.py — Portfolio Manager (риски, корреляции)

START.bat функционал:
[1] Запуск всего (Backend + Frontend + Live Engine)
[2] Только Backend API
[3] Только Frontend
[4] Live Engine (торговля)
[5] Установка зависимостей (Python + Node)
[6] Скачивание истории свечей
[7] Инициализация БД
[8] Запуск тестов
[9] Проверка статуса

Интеграция:
- Backend API (FastAPI) на порту 8000
- Frontend (Vite) на порту 5173
- SQLite база данных в data/velas.db
- WebSocket для real-time обновлений

Следующие шаги:
- WebSocket интеграция frontend ↔ backend
- E2E тестирование
- Production оптимизация
